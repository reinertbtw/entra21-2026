Este repositório contém os projetos, exercícios e atividades desenvolvidos durante o curso técnico de Desenvolvimento de Sistemas do programa Entra21, com foco em Java e desenvolvimento de software.

## Sobre o curso

O Entra21 é um programa de capacitação profissional que tem como objetivo formar desenvolvedores preparados para o mercado de trabalho na área de tecnologia.

Durante o curso, são abordados desde fundamentos de programação até desenvolvimento web, banco de dados e frameworks modernos.

## Módulos do curso

- Módulo 01 – Fundamentos de Programação com Java  
- Módulo 02 – Metodologias Ágeis para TI  
- Módulo 03 – Git e GitHub: Controle de Versão Simples  
- Módulo 04 – Fundamentos de HTML e CSS  
- Módulo 05 – Introdução ao JavaScript  
- Módulo 06 – Introdução a Banco de Dados  
- Módulo 07 – Programação Orientada a Objetos (OOP)  
- Módulo 08 – Java para Web: Fundamentos  
- Módulo 09 – Introdução ao Spring Framework  
- Módulo 10 – Tecnologias Emergentes e IA  

## Objetivo

- Documentar minha evolução no curso Entra21  
- Praticar lógica de programação e desenvolvimento de software  
- Aprender e aplicar Java e tecnologias relacionadas  
- Construir um portfólio técnico para o mercado de trabalho  
- Praticar boas práticas com Git e GitHub
