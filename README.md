## Sobre o curso

O Entra21 é um programa de capacitação profissional voltado à formação de desenvolvedores de software, com o objetivo de preparar estudantes para o mercado de trabalho na área de tecnologia.

Durante o curso, são abordados desde fundamentos de programação até desenvolvimento web, banco de dados, frameworks modernos e boas práticas de desenvolvimento.

---

## Módulos do curso

- Módulo 01 – Fundamentos de Programação com Java  
- Módulo 02 – Metodologias Ágeis para TI  
- Módulo 03 – Git e GitHub: Controle de Versão Simples  
- Módulo 04 – Fundamentos de HTML e CSS  
- Módulo 05 – Introdução ao JavaScript  
- Módulo 06 – Introdução a Banco de Dados  
- Módulo 07 – Programação Orientada a Objetos (OOP)  
- Módulo 08 – Java para Web: Fundamentos  
- Módulo 09 – Introdução ao Spring Framework  
- Módulo 10 – Tecnologias Emergentes e IA
- Módulo 11 – Desenvolvimento do Trabalho de Conclusão
